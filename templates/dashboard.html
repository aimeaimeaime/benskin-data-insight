Observatoire Intelligent du Transport Urbain

🚀 Contexte du Projet : Secteur des Moto-Taxis

Cette application est conçue pour moderniser le secteur informel du transport urbain (**Benskin**). L'objectif est d'utiliser l'Intelligence Artificielle pour transformer les données collectées sur le terrain en outils de décision.

Elle permet de :

  • Optimiser les revenus des conducteurs via la Régression.
  • Améliorer la sécurité routière en prédisant les risques d'accidents (Classification).
  • Segmenter les profils pour des politiques sociales ciblées (Clustering).

Méthode : Algorithme entraîné pour prédire la probabilité d'accident selon l'expérience et l'âge.
